# 对话助手应用

这是一个基于Flask和OpenAI API（DeepSeek模型）的对话助手应用，提供简洁而强大的对话界面，支持历史对话管理和AI思考内容的显示/隐藏功能。

程序参数详解请点击此处[README_2](./README_2.md)

## 功能特点

- 基于DeepSeek模型的智能对话功能
- 支持多轮对话，自动保存对话历史
- 流式响应，提供实时生成效果
- 显示/隐藏AI思考过程
- 支持Markdown格式显示
- 可配置API密钥和服务地址
- 响应式设计，适配移动设备

## 安装与部署

### 前提条件

- Python 3.6 或更高版本
- pip 包管理工具

### 安装步骤

1. 克隆或下载本项目到本地

2. 安装依赖

   ```bash
   cd deepseekapi
   pip install -r requirements.txt
   ```

3. 启动应用

   ```bash
   python app.py
   ```

4. 在浏览器中访问应用

   打开 `http://127.0.0.1:5000` 即可使用对话助手

## 使用方法

### 首次使用配置

1. 点击右上角的设置图标（⚙️）
2. 输入DeepSeek API的密钥（API Key）和基础URL（Base URL）
3. 点击「保存设置」完成配置

### 基本对话

1. 在输入框中输入您的问题或指令
2. 点击「发送」按钮或按下Enter键提交
3. AI会实时生成回答，支持流式显示

### 历史对话管理

- **创建新对话**：点击侧边栏中的「新对话」按钮
- **切换对话**：点击侧边栏中的对话项
- **删除对话**：点击对话项右侧的删除按钮（🗑️）

### 显示/隐藏AI思考

- 点击顶部导航栏的「显示思考」按钮可查看AI的思考过程
- 再次点击「隐藏思考」按钮可隐藏思考内容

## 注意事项

- 请妥善保管您的API密钥，避免泄露
- 免费API通常有使用限制，请合理使用
- 如遇到连接问题，请检查网络设置和API配置
- 应用默认在本地运行，如需外部访问，请配置相应的网络设置

## 技术栈

- 后端：Flask
- 前端：HTML, JavaScript, Tailwind CSS
- API集成：OpenAI API 兼容接口
- 数据存储：JSON文件
